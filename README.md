# Portfolio

A simple portfolio website using a React, and Gatsby.




# How to Use
1. Clone or Download repo.
2. cd into folder.
3. Run  ``` npm install ```.
4. Run  ``` npm start ```.
5. Go to local dev port 8000 (http://localhost:8000)
6. Make changes in src/ 




## Built with

- JavaScript
- Gatsby
- React 
- VSCode
